# Pythia_cc
Phenomenological study of Intrinsic Charm signatures in p-p collisions at 13.6 TeV using Pythia 8. Focused on the forward kinematic region (LHCb acceptance).

The `DD.ipynb` notebook contains a pipeline for simulation of the heavy-flavor events. Specifically, it covers:

* Setting up the required environment and downloading the necessary PDF sets (CT18NNLO and NNPDF4.0).
* Generating Monte Carlo events using Pythia 8 and filtering for D0 and anti-D0 mesons within the defined forward acceptance region.
* Processing the generated data to visualize key kinematic distributions, including transverse momentum, rapidity, and total cross-sections.
